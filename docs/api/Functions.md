## Get_Event_Name

根据 id 获取特定的事件名称

- 定义

```php
Get_Event_Name($event)
```

- 参数

| 名称   | 类型   | 默认值 | 描述    |
| ------ | ------ | ------ | ------- |
| $event | string | N/A    | 事件 id |

- 返回值

| 名称 | 类型   | 描述   |
| ---- | ------ | ------ |
| N/A  | string | 事件名 |
